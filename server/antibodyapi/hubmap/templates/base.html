Antibody Validation Report

The HuBMAP Antibody Validation Report Upload allows new reports to be uploaded to the HuBMAP AVR tracking system.

To add reports, upload a csv file as described in the AVR upload documentation formatted based on the template csv file along with all of the AVR reports referenced in the csv as PDFs.

Upload a single csv file describing the AVRs being uploaded (see AVR upload documentation and template csv file)

Upload the PDF files associated with the csv by clicking Browse. To easily upload place all files in a single directory and use the ctrl (or command on Mac) key to select them all in the file dialog.
